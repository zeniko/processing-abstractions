
## Installation

```st
Metacello new
	repository: 'github://zeniko/processing-abstractions:main/src';
	baseline: 'ProcessingAbstractions';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfProcessingAbstractions asClass loadLepiter
```
